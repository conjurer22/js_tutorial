Different git status as per the actions- 1) modified- changes made to the file that have not been updated on github. 2) untracked- a new file, previously not available on github, for the same project has been added. 3) staged- file or changes are ready to be committed, ie changes will be added to the github repo too. 4)unmodified- after staging when the committs are finally made, the status changes to unmodified aka unchanged.
